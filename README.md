## How to use

- Add Tsumego Problems
Click 'File' - 'Manage Tsumego' - 'Add' (bottom left) - 'Add Existing Files...' (add multiple tsumego sgf files)

- Enable Tsumego Mode
Click 'Tools' -  'Toggle Tsumego Mode' - Enjoy solving tsumego problems!
